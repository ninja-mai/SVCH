## 🎓 Курсовая работа

### Разработка фазированной антенной решётки (ФАР)
[📄 kursovaya.pdf](./kursovaya.pdf)

**Тема:** Проектирование ФАР на щелевых резонаторных излучателях

| Параметр | Значение |
|----------|----------|
| Рабочая частота | 12 ГГц |
| Ширина ДН (0,5 мощности) | 2° |
| УБЛ | < 18 дБ |
| Макс. угол сканирования | 20° |
| Мощность передатчика | 10 кВт |
| Конфигурация решётки | 32×16 (512 излучателей) |
| Тип излучателя | Щелевой резонаторный |

**Что сделано:**
- Расчёт одиночного щелевого излучателя
- Выбор амплитудного распределения (косинус на пьедестале 0,4)
- Расчёт ДН одиночного элемента и всей решётки
- Сравнение конфигураций 32×32 и 32×16
- Проектирование схемы питания (коаксиальная линия 50 Ом)
- Оценка потерь в проводнике и диэлектрике
